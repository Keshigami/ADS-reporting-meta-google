## Data Sources for Report

| Section | Data Source | Metric |
|---------|-------------|--------|
| Google Ads metrics | Google Ads API | spend, clicks, conversions |
| Google Ads calls | Google Ads API | phone_calls, phone_impressions |
| Facebook metrics | Meta Ads API | spend, clicks, leads |
| Facebook calls | GHL API | call count from GHL number |

---

## How n8n Generates This

```
1. Pull Google Ads data (including phone_calls metric)
2. Pull Meta Ads data
3. Pull GHL call logs (for Facebook tracking number)
4. Merge data by client
5. Generate Google Doc from template
6. Export to PDF
7. Email to client + upload to Drive
```

---

*This template is used by the n8n workflow to generate weekly reports.*
