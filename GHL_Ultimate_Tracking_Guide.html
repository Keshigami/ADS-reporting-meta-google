Ultimate Tracking & Attribution Guide

Strategy is the Blueprint. This is the technical execution.

Objective: Consolidate all tracking protocols into one workflow. Follow these steps to ensure every dollar spent in Google and Meta is accurately tracked to a Lead and Sale in HighLevel.

01 / The Foundation

Before connecting ads, you must understand how the platform attributes revenue.

First vs. Latest Attribution

  • First Attribution: The verified first interaction (e.g., SEO visit 6 months ago). Never changes.
  • Latest Attribution: The most recent interaction (e.g., Facebook Ad click yesterday). Updates constantly.

Where to Find It

Go to Contacts > Select Contact > Scroll to Activity (Bottom Right).

02 / Google Ads Setup

2.1 Connect the Account

  1. Go to Settings > Integrations.
  2. Connect your Google Account (Must have Admin access).
  3. Go to Reporting > Google Ads Settings.
  4. Select your MCC Account (if applicable) and Client Account.
⚠️ Troubleshooting: Ensure the user has "Admin" access in Google Ads and "User" or "Admin" access in GHL. Manager roles can cause disconnects.

2.2 Install the Tracking Template (CRITICAL)

For Google Ads to "talk" to the CRM, you must use this specific UTM template. This is the bridge between ad spend and revenue.

{lpurl}?utm_source=google&utm_medium=cpc&utm_campaign={campaignname}&utm_content={adgroupname}&utm_term={keyword}&campaign_id={campaignid}&ad_group_id={adgroupid}&ad_id={creative}

Where to Paste It:

  • Google Ads > Settings > Account Settings > Tracking.
  • Paste into "Tracking template".

2.3 Enable Auto-Tagging

Go to Google Ads > Settings > Account Settings > Auto-tagging and ensure it is ON. This automatically appends the gclid to URLs for Enhanced Conversions.

2.4 The "Fail-Safe" Script (Optional)

Ensure 100% data capture even if UTMs fail. Go to Google Ads > Tools > Scripts and create "GHL Fail-Safe".

03 / Facebook Ads Setup

3.1 Connect & Enable

  1. Go to Settings > Integrations > Connect Facebook.
  2. Go to Reporting > Facebook Ads Settings.
  3. Select the correct Ad Account.

3.2 Paid Social Parameters

utm_source=facebook&utm_medium=paid_social&utm_campaign={{campaign.name}}&utm_content={{ad.name}}&utm_term={{adset.name}}&campaign_id={{campaign.id}}

Where to Paste It: Url Parameters field at the Ad Level.

04 / Advanced Tracking (GA4)

While GHL has its own reporting, Google Analytics 4 (GA4) is the verified "Source of Truth" for traffic.

4.1 Tracking Funnel Pageviews in GA4

  1. In GA4, go to Admin > Property Settings > Data Streams.
  2. Copy the Global Site Tag (G-XXXXXXXX).
  3. In GHL, paste this into Funnel Settings > Head Tracking Code.

4.2 Sending Workflow Data to GA4

You can send "Offline" events (like a manually added lead) to GA4 via Workflows.

  1. Create a Workflow > Add Action: "Google Analytics".
  2. Enter your Tracking ID.
  3. Define Event Category (e.g., "Lead"), Action, and Label.

06 / Advanced Optimization

Conversion Workflows (Server-Side Tracking)

Create these 4 workflows to send conversion data back to Google and Meta for ROAS calculation:

Workflow Name Trigger Action
Google Ads - Track Lead Contact Created Add to Google AdWords (Lead)
Google Ads - Track Closed Won Pipeline → Closed Add to Google AdWords (Closed Won)
Meta Ads - Track Lead Contact Created Facebook CAPI (Lead)
Meta Ads - Track Purchase Pipeline → Closed Facebook CAPI (Purchase)
⚠️ Critical: For Revenue to work, every closed opportunity MUST have a monetary value set!

Call Tracking Setup

Use both methods for complete coverage.

Option A: Static Tracking Numbers (For Ads)

Path: Settings > Phone Numbers > Buy Number

Channel Number Label Where to Use
Google Ads CLIENT-GOOGLE Google call extensions, search ads
Meta Ads CLIENT-META Facebook/Instagram ad creatives
LSA CLIENT-LSA Local Service Ads profile
Organic CLIENT-ORGANIC GMB, website footer, direct

Cost: ~$6-8/month (3-4 numbers × $2/each)

Option B: Number Pools (Dynamic Website Tracking)

Number Pools use Dynamic Number Insertion (DNI) to swap phone numbers at the visitor level for keyword-level call attribution.

Path: Settings > Phone Numbers > Number Pool > Add Number Pool

  1. Tracking Type: Select "Visitors Activity"
  2. Pool Size: Minimum 4 numbers
  3. Forwarding Number: Client's main business line
  4. Swapping Number: Number displayed on website

After creating the pool, click Get Code and paste the snippet into Footer Tracking Code of your funnel.

⚠️ Important: The swap script must be in the Footer (body), not the Header. Test in incognito browser to verify swapping works.

External Site Tracking (WordPress/Wix)

For clients with external websites, install the GHL tracking script:

Path: Settings > Business Profile > Tracking Code

  1. WordPress: Install "Insert Headers and Footers" (WPCode) plugin
  2. Paste GHL tracking script in Header section
  3. Save changes

Trigger Links (Email/SMS Attribution)

Track clicks in emails and SMS with automated workflow triggers:

Path: Marketing > Trigger Links > + Create

  1. Create trigger link with destination URL
  2. Insert in email/SMS campaigns
  3. Create workflow with "Trigger Link Clicked" trigger

Email Tracking Setup

Path: Settings > Email Services > Dedicated Domain

  1. Add dedicated sending domain (e.g., mail.clientdomain.com)
  2. Configure DNS: SPF, DKIM, DMARC
  3. Enable Click & Open Tracking

Offline Conversions (The Feedback Loop)

  1. Google Ads: Create Conversion Action "Import from Clicks".
  2. Workflow: Trigger "Form Submit" -> Action "Add to Google Analytics/Ads".

Client Spend Markup

Agency Settings > Reporting > Client Spend Markup. Hides raw costs and displays (Ad Spend + Management Fee) to the client.
